# 坦克大战游戏项目开发计划

##  Context（项目背景）

**项目目标**：使用 Python 开发一款画面精美的坦克大战游戏，还原经典坦克大战游戏的操作体验和游戏玩法。

**项目背景**：经典的坦克大战（Battle City）是一款深受喜爱的街机游戏。本项目旨在通过 Python 和 Pygame 框架重现这一经典，同时加入现代化的视觉效果。

---

## 核心功能模块

### 1. 游戏主循环（game.py）
- 初始化 Pygame 和显示窗口（1280x720）
- 处理游戏状态（菜单/游戏中/暂停/结束）
- 控制帧率（60 FPS）
- 事件处理（键盘/鼠标）

### 1.5. 配置系统（config.py）
- 游戏分辨率配置（1280x720）
- 简体中文语言配置
- 游戏速度/难度设置

### 2. 坦克系统（tank.py）
- **玩家坦克**：
  - WASD 或方向键控制移动
  - J 键或空格键射击
  - 支持升级系统（速度/火力/防御）
  - 生命系统（默认 3 条命）
- **敌人坦克**：
  - AI 控制移动和射击
  - 多种敌人类型（普通/快速/重装/ Boss）
  - 生成机制

### 3. 子弹系统（bullet.py）
- 子弹类型（普通/高速/穿甲）
- 飞行轨迹计算
- 伤害计算

### 4. 碰撞检测（collision.py）
- 坦克与子弹碰撞
- 坦克与地形碰撞
- 子弹与地形碰撞
- 坦克间碰撞

### 5. 地形系统（terrain.py）
- 可破坏砖块
- 不可破坏钢板
- 水域（阻挡坦克，子弹可飞过）
- 草地（隐藏坦克）
- 基地（老鹰/城堡，被摧毁则游戏结束）

### 6. 关卡管理（level.py）
- 10 个精心设计的关卡
- 关卡地图加载
- 敌人波次生成
- 关卡难度递增

### 7. UI 系统（ui.py）
- 主菜单界面（简体中文）
- 游戏 HUD（生命值/关卡/得分）
- 暂停菜单
- 游戏结束界面
- 中文字体渲染支持
